# AI Video Cameras - KlingO1 运镜合集

本项目是一个专门收集和展示 AI 生成视频（基于 KlingO1 模型）运镜效果的资源库。它旨在为 AI 视频创作者提供直观的运镜参考，并提供标准化的元数据和多分辨率视频文件。

## 1. 仓库价值

*   **运镜参考指南**：涵盖了从基础推拉摇移到高级的“极速俯冲”、“液态变形”等 50+ 种运镜效果。
*   **标准化资源**：每个运镜均配有：
    *   **1080p 源文件**：高质量原始素材。
    *   **720p 预览版**：经过 H.264 优化，适合网页快速加载。
    *   **精准 Prompt**：提供生成该效果的英文提示词，方便复现。
*   **即时预览**：内置基于 HTML5 的展示页面，支持悬停自动播放，快速对比不同运镜差异。

## 2. 视频归档逻辑与技术流程

为了确保视频在不同设备上的兼容性及网页加载性能，本项目遵循以下归档流程：

### 步骤 A：视频下载与命名
使用 `curl` 获取原始 1080p 视频，并统一命名规范：`klingO1_[运镜英文名].mp4`。

### 步骤 B：720p 转换 (ffmpeg)
由于源视频多为竖屏（1080x1920），直接缩放可能导致宽度不被 2 整除而报错。我们使用 `-2` 参数自动计算偶数宽度。
```bash
ffmpeg -i input_1080p.mp4 -vf "scale=-2:720" -c:v libx264 -crf 23 -preset fast output_720p.mp4
```
*   参数说明：
    *   `-vf "scale=-2:720"`：固定高度为 720px，宽度按比例缩放并自动调整为最接近的偶数。
    *   `-c:v libx264`：使用 H.264 编码，确保浏览器兼容性。

### 步骤 C：缩略图提取
从视频第 1 秒处提取一帧作为封面图：
```bash
ffmpeg -i input.mp4 -ss 00:00:01 -vframes 1 thumbnail.jpg
```

### 步骤 D：元数据同步
更新 `videos/klingO1/videos.json`，记录视频的中文标题、英文标题、Prompt、路径及分辨率信息。
